eh noises

Eh Noises is a FOSS-based noise album.

I used only Free and Open Source Software (FOSS): Pure Data, Arduino IDE, Ardour, on a Linux computer running the EndeavourOS distribution. I also used a violin, a cello, a nylon guitar, and a DYI Arduino controller.
