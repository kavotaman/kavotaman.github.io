Rip VS-1680 backup cds to PC

  • Download VS Wave Export tool
  • Needs to have Windows 10
  • Unzip VSWE
  • Right click program file, go to compatibility tab, check “Run as admin”
  • Copy cd2roland exe to some place like C:\test
  • Open VSWE
  • Click “Extras” on menu; click CD rip (or something like that)
  • There, it has a field already filled. This needs to be changed a bit
    • You need to specify where the command cd2roland is. So prefix the command with the place you put the cd2roland.exe. In our case, C:\test\cd2roland
    • The capital letter after the command (cd2roland) is the drive where the backup CD is. As default it shows D. If your CD drive is on E:, change the letter to E.
    • Click “run”
  • If everything works, you’ll get a command prompt window that will run some processes. Press any key when prompted
  • You should have a list of files in the main window of the VSWE program, called Projects.
  • If the project has files, click “All”, then “Export”.
  • After you’re done with everything, I recommend closing the program and erasing the folder Rip0000 before inserting a new backup CD.

Zero padding

If you have files named lala1.txt lala2.txt … lala23.txt etc and want to add zeroes so that it’s better do:

create bash script:

#!/bin/bash
num=`expr match "$1" '[^0-9]*\([0-9]\+\).*'`
paddednum=`printf "%03d" $num`
echo ${1/$num/$paddednum}

on a terminal, where the files are, type: for i in *.jpg;do mv $i '~/Scripts/zeropad.sh $i'; done

change the *.jpg to the extension of your liking.
